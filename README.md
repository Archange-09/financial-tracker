# financial-tracker
An expense tracker program that uses Random Forest for predicting future expenses.
##  Description
* **Expense Prediction:** Uses a Random Forest model to forecast next month's spending.
* **Excel Export:** Save your data into `.xlsx` files with one click.
* **Modern Interface:** Sleek design using the Forest-ttk theme.
* **Charts & Graphs:** Visualizes your spending trends over the year.

* How to Run
1. Clone the project:
   git clone [https://github.com/Archange-09/financial-tracker.git](https://github.com/Archange-09/financial-tracker.git)
2. Install the Dependencies
   pip install pandas scikit-learn matplotlib openpyxl
3. Launch the app
